# Setups

Cada archivo `.html` de esta carpeta es la **guía 3D interactiva de un setup** del mapa.

Son páginas HTML autocontenidas (se abren directo en el navegador). Cada una explica un
setup paso a paso: qué misión reclamar, qué craftear, con qué mods se combina, cómo
automatizarlo y cómo conectarlo al almacenamiento digital — con escenas 3D rotables y
tooltip por bloque.

Conforme se van creando, se enlazan desde el mapa maestro (`../index.html`) cambiando el
estado del nodo a *Disponible*.
